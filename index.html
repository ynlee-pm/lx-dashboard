Apps Script 배포 URL 권장
🔧 Apps Script 1회 설정 방법 (CORS 오류 해결)

로컬 HTML 파일에서 Google Sheets를 직접 불러오면 브라우저 보안 정책(CORS)으로 차단됩니다.
Apps Script 웹 앱을 한 번만 설정하면 이후 버튼 하나로 동작합니다.

  1. 스프레드시트 열기 → 확장 프로그램 → Apps Script
  2. 기존 코드 전체 삭제 후 아래 코드 붙여넣기 → 💾 저장
  3. 상단 배포 → 새 배포 클릭
  4. 유형: 웹 앱 / 액세스: "모든 사용자" 선택 → 배포
  5. 표시된 웹 앱 URL 복사 → 위 입력창에 붙여넣기 → 저장
  6. 아래 데이터 불러오기 버튼 클릭 ✅
function doGet(e) {
  const ss    = SpreadsheetApp.openById('101oIzi_iNX4cSMrfq-va4T2mJRWxroyYweCgJDbhU2k');
  const sheet = ss.getSheets().find(s => s.getSheetId() === 1599933316) || ss.getSheets()[0];
  const data  = sheet.getDataRange().getDisplayValues();
  const csv   = data.map(row =>
    row.map(cell => {
      const s = String(cell);
      return (s.includes(',') || s.includes('"') || s.includes('\n'))
        ? '"' + s.replace(/"/g, '""') + '"' : s;
    }).join(',')
  ).join('\n');
  return ContentService.createTextOutput(csv)
    .setMimeType(ContentService.MimeType.TEXT);
}
